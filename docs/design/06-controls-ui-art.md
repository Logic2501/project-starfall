# 06. 操作、UI 与程序生成美术

## 1) 输入与交互

| 输入 | 功能 |
| --- | --- |
| 鼠标移动 | 瞄准主炮塔 |
| 左键点击/按住 | 开火 / 持续激光 |
| 波间 UI 点击 | 切换主炮塔模式 |
| 波间 UI 点击 | 切换火箭协议 |
| 科技树 UI 点击 | 解锁科技 |

### 切换规则

- 主炮塔模式：仅波间允许
- 火箭发射台协议：建议仅波间允许
- 波内切换：不支持

## 2) HUD / UI 需求

- 顶部：当前波次、波次预告、暂停
- 左上：资源点
- 右上：主炮塔模式、火箭协议
- 中央上方：来袭预警箭头、目标标签
- 波后面板：分解率、捕获率、保存率、资源收益
- 波间面板：模式切换、协议切换、科技树入口

## 3) 程序生成美术规范

### 风格

- Canvas 渲染
- 低分辨率、整数缩放、限定色板
- 尽量不用位图，优先几何体/噪声/粒子

### 对象生成策略

| 对象 | 生成方式 |
| --- | --- |
| 星球弧面 | 圆弧 + 噪声地表 + 分层色带 |
| 大气层 | 半透明渐变环 |
| 小行星 | 不规则多边形 + 表面斑点 + 自转 |
| 碎块 | 小多边形 + 速度向量 + 生命周期 |
| 火炮特效 | 炮口闪光 + 弹道线 + 爆裂环 |
| 激光特效 | Beam 线段 + 命中发光点 + 切割分离 |
| 火箭特效 | 小导弹体 + 尾焰粒子 |
| 资源回收 | 下落光点 + 落地闪光 + 计数跳字 |
