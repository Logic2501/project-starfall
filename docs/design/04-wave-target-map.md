# 04. 波次、目标与地图

## 1) 波次预告

### 预告信息维度

- 规模：小型群 / 中型混合 / 大型单体 / 稀有高价值目标
- 速度：低速 / 中速 / 高速
- 材质：脆性 / 岩质 / 金属富集 / 冰岩混合
- 风险：易飞逸 / 易烧蚀 / 大气衰减显著 / 捕获窗口短

### 预告示例

- 第4波：脆性小型体群 / 低速 / 低价值
- 第7波：岩质中型体 / 中速 / 捕获窗口短
- 第11波：金属富集体 / 中速 / 高价值
- 第15波：大型单体 / 高速 / 厚壳
- 第19波：小型高速碎片雨 / 易飞逸

### 波间可调整项

- 主炮塔模式：常规火炮 / 高级火炮 / 火箭助推炮弹 / 制导导弹 / 激光切割
- 火箭协议：大块优先 / 完整目标优先 / 高价值优先 /（后续）高速修正优先

## 2) 目标系统（v1最少4类）

| 类型 | 特征 | 推荐处理 |
| --- | --- | --- |
| 小型岩质体 | 易碎、价值低 | 常规火炮 |
| 金属富集体 | 不易碎、价值高 | 激光切割 / 制导导弹 |
| 冰岩混合体 | 易散损耗高 | 完整目标优先回收 / 轻处理 |
| 大型缓慢目标 | 价值高、可整块处理 | 制导导弹 / 火箭发射台 |

目标基础属性：基础价值、结构强度、脆性、速度等级、大小等级、材质标签。

## 3) 地图设计

### 地图1：稀薄大气（教学）

- 常规火炮可用
- 回收难度低
- 学习“打碎-掉落-转资源”和火箭发射台意义

### 地图2：无大气（中期）

- 激光表现好
- 自然捕获率低、飞逸风险高
- 学习“不是越碎越好，大块保留更重要”

### 地图3：稠密大气（后期）

- 火炮与激光衰减明显
- 火箭助推炮弹/制导导弹更关键
- 再入烧损显著，需要再入保护技术
