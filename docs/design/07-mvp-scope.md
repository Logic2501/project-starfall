# 07. MVP 范围与开发优先级

## 1) 第一版必须实现（MVP）

- 地图：3张（稀薄大气 / 无大气 / 稠密大气）
- 设施：1主炮塔 + 1火箭发射台
- 主炮塔模式：常规火炮 / 高级火炮 / 火箭助推炮弹 / 制导导弹 / 激光切割
- 火箭协议：大块优先 / 完整目标优先 / 高价值优先
- 目标类型：小型岩质体 / 金属富集体 / 冰岩混合体 / 大型缓慢目标
- 科技树：4条线，约20~24节点
- 资源：单一通用资源点
- 结算：分解率 / 捕获率 / 保存率 / 综合回收效率

## 2) 暂缓实现

- 生态建设系统
- 多资源类型
- 复杂轨道力学
- 多塔布局
- 火箭炮 / 电磁辅助发射（v2）

## 3) 实现优先级

### P1：核心玩法打通

1. 波次系统
2. 目标生成与飞行
3. 主炮塔基础开火
4. 命中后碎裂/切割
5. 简化捕获与落地结算
6. 资源结算与增加
7. 波间切换模式

### P2：方案感建立

1. 五种主炮塔模式差异
2. 火箭发射台自动回收协议
3. 三张地图环境修正
4. 波次预告 UI

### P3：长期循环

1. 科技树前置解锁
2. 波后数据面板
3. 适用性调参与平衡
