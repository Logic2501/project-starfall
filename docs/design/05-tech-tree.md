# 05. 科技树设计

## 1) 总体原则

- 科技树是能力展开树，不是纯数值树。
- 目标是解锁“新方法与新适用域”。
- 制导技术是先进火炮/导弹前置。
- 旧技术在特定场景仍可用。

## 2) 分支总览

- 火炮基础线：基础发射能力与火炮模式
- 制导技术线：中高速目标处理与导弹前置
- 激光技术线：高保留率切割
- 回收与再入线：发射台能力与地图3保存率

## 3) 火炮基础线

| 节点 | 前置 | 效果 |
| --- | --- | --- |
| 黑火药装药 | 无 | 解锁基础火炮概念 |
| 金属炮管 | 黑火药装药 | 解锁常规火炮 |
| 无烟火药 | 金属炮管 | 提升稳定性与射速 |
| 膛线加工 | 无烟火药 | 提升精度 |
| 高膛压结构 | 膛线加工 | 解锁高级火炮 |
| 火箭助推炮弹 | 高膛压结构 | 解锁对应模式并降低厚大气衰减 |

## 4) 制导技术线

| 节点 | 前置 | 效果 |
| --- | --- | --- |
| 光学测距 | 无 | 提升基础命中辅助 |
| 预测射击 | 光学测距 | 改善中高速目标处理 |
| 姿态修正 | 预测射击 | 为制导武器与发射台提供前置 |
| 基础制导 | 姿态修正 | 导弹类技术前置 |
| 末段制导 | 基础制导 | 解锁制导导弹模式 |
| 高速目标解算 | 末段制导 | 提升高速目标有效处理率 |

| 节点 | 前置 | 效果 |
| --- | --- | --- |
| 聚焦透镜 | 无 | 激光技术前置 |
