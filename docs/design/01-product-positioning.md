# 01. 项目定位与核心玩法

## 1) 项目定位

| 项目 | 内容 |
| --- | --- |
| 暂定类型 | 工程方案选择型二维塔防 / 资源回收游戏 |
| 平台 | 浏览器（PC 优先） |
| 画面 | 纯程序生成美术、低分辨率、限定色板、Canvas 渲染 |
| 核心灵感 | Green the Planet（打天体拿资源）+ KSP（按环境选技术方案） |
| 非目标 | 非 roguelike build 取舍；非重拟真轨道模拟；非漏怪扣血塔防 |
| 核心乐趣 | 预读波次 -> 识别目标 -> 波间切换主炮塔 -> 火箭回收提效 |

## 2) 一句话核心玩法

玩家在不同星球环境中，根据每波预告的小行星类型，在波间切换主炮塔技术方案，并利用火箭发射台主动回收高价值目标或大块残骸，以最大化资源回收效率。
